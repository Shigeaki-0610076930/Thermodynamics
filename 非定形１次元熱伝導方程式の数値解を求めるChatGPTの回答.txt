以下の条件で非定形１次元熱伝導方程式の数値解を求めるpythonコードを提示願います。

[方程式]
\frac{\partial T}{\partial t} = a \frac{\partial^2 T}{\partial x^2}
T: 温度
t: 時間
x: 座標
a: 温度伝導率

[座標xの変化]
i=0,1,2,3,4,5

[初期条件]
t=0, T[i]=20(℃)

[境界条件]
t>0, T[0]=0(℃), T[5]=20(℃)

[温度伝導率]
a=0.1

[時間刻み幅]
⊿t=5(秒)

[プログラミング]
陽解法で、 ０秒から安定する時刻まで、各座標T[i]の値をプリントアウトしてください。

以下は、与えられた条件に基づいて非定形１次元熱伝導方程式の数値解を求めるためのPythonコードです。このコードは陽解法を使用しています。


import numpy as np

# パラメータ設定
a = 0.1  # 温度伝導率
dt = 5  # 時間刻み幅
dx = 1  # 空間刻み幅

# グリッドの設定
nx = 6  # 座標点の数
nt = 100  # タイムステップ数

# 初期条件の設定
T = 20 * np.ones(nx)
T[0] = 0
T[-1] = 20

# 陽解法による数値解法
for n in range(nt):
    Tn = T.copy()
    for i in range(1, nx - 1):
        T[i] = Tn[i] + a * dt / dx**2 * (Tn[i+1] - 2*Tn[i] + Tn[i-1])

    # 安定時刻までの各座標T[i]の値をプリントアウト
    print(f"Time step {n}: {T}")

このコードでは、与えられた条件に基づいて、空間および時間の刻み幅を定義し、グリッドを作成しています。
次に、初期条件を設定し、陽解法を使用して時間ステップごとに非定形熱伝導方程式の数値解を計算しています。
計算結果は時間ステップごとにプリントアウトされます。
